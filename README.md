Automatización-Ciberseguridad

Este proyecto ha sido desarrollado para la automatización de tareas de ciberseguridad mediante la integración de múltiples tecnologías como Bash, PowerShell y Python, permitiendo la ejecución coordinada de procesos de análisis, auditoría y generación de reportes

¿Qué contiene? Una solución automatizada organizada en módulos que combina scripts de diferentes entornos para realizar tareas de reconocimiento de red, auditoría del sistema y análisis de información, todo centralizado desde un ejecutable principal en Python

¿Qué scripts se incluyen?
Módulo 1 (Reconocimiento de Red) un script en Bash que consulta información pública de un dominio o dirección IP y almacena los resultados en formato JSON
Módulo 2 (Auditoría del Sistema) un script en PowerShell que obtiene información relevante del sistema como procesos activos para identificar posibles riesgos
Módulo 3 (Análisis y Reporte) un script en Python que procesa la información recolectada, genera hashes de integridad y construye un reporte final en HTML

¿Qué tareas de ciberseguridad resuelven?
Obtención de inteligencia básica de red mediante consultas a servicios externos
Monitoreo del estado del sistema a través del análisis de procesos activos
Verificación de integridad de datos mediante el uso de hashing (MD5)
Automatización del flujo de análisis reduciendo intervención manual

¿Qué aprendí al desarrollarlo?
Aprendí a integrar diferentes lenguajes de programación dentro de un mismo flujo de trabajo, así como a automatizar procesos utilizando Python como orquestador principal
También reforcé el manejo de estructuras de datos en JSON, ejecución de comandos del sistema y generación de reportes automatizados

Durante el desarrollo de este proyecto se fortalecieron las habilidades en automatización, scripting y organización de código, logrando una integración eficiente entre distintas herramientas y asegurando la trazabilidad mediante registros de logs, lo que permite identificar errores y dar seguimiento a cada ejecución realizada
